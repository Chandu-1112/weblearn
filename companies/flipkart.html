What Flipkart Wants from Freshers as Web Developers

  • Proficiency in HTML, CSS, JavaScript (ES6+).
  • Knowledge of building high-performance e-commerce UIs.
  • Hands-on experience with React.js or Vue.js is highly preferred.
  • Familiarity with APIs, JSON data, and AJAX.
  • Good sense of UX and UI design for fast-loading pages.
  • Experience with performance tools like Lighthouse or PageSpeed Insights.
  • Familiarity with Git and CI/CD workflows.
  • Passion for coding and eagerness to learn new technologies.
